LocusMapTweak
=============
These files extend the available online map providers in Locus Map. 
The `providers.xml` file is the essential component. Icons are optional and just for decoration.

**Note:** This project is no longer actively maintained. The existing files remain available for use, but please understand that support, updates, and questions cannot be accommodated. Feel free to fork and modify as needed!

Do it yourself
--------------
Copy the file `providers.xml` (found in this repository) and all icons into the following directory on your device:<br>
`...\Locus\mapsOnline\custom`

One-click installation
----------------------
For a convenient installation method, visit this website **directly from your Android device** with Locus Map installed: <br>
https://sites.google.com/view/locusmaptweak/locus-map-tweak

The website allows you to install the additional map providers with a single click. This only works when accessed from the Android device where Locus Map is installed.

Contribute to providers.xml
---------------------------
Find or fix maps using this method: <br>
http://docs.locusmap.eu/doku.php?id=manual:advanced:customization:online_maps
